02 / WHAT CHANGED

The skill grew teeth.

These are not more adjectives for the agent to imitate. They are scripts, assets, files, and checks the agent can actually run.

A different palette every project

Five moods guide hue ranges and type pairings, but every color system is generated fresh and solved against the contrast floor.

The script samples the source image itself, then returns exact colors instead of a vague description.

Taste survives the session

A project-level style lock prevents drift. A developer profile learns only from real keep/reject decisions, never fabricated approval.

Assets in the first pass

Photos, coherent icons, illustrations, a constructed logo, and favicons arrive with the screen.

Motion is part of the build

Landing pages get scroll storytelling. App shells get state transitions and loading motion.

Visuals do the explaining

Product mockups, flows, and comparisons replace the familiar wall of feature copy.
